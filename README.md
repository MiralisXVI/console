# Aplicación JavaScript en Consola 

## Funcionalidades
- **Operaciones Matemáticas**: Suma, resta, multiplicación y división con validación de entradas.
- **Historial de Operaciones**: Almacena y muestra el historial de operaciones realizadas.
- **Trabajo con Arreglos**: Crea un arreglo, lo filtra y lo recorre con diferentes bucles.
- **Información del Usuario**: Maneja datos del usuario usando objetos con métodos.

## Estructura del Código
- **Variables**: Uso de `let` y `const` para declarar variables.
- **Condicionales**: `if`, `else`, `switch` para control de flujo.
- **Bucles**: `for`, `while`, `forEach` para iteraciones.
- **Funciones**: Modularización del código en funciones reutilizables.
- **Arreglos**: Manipulación de datos con `filter`, `push`, etc.
- **Objetos**: Creación de objetos con propiedades y métodos.

## Cómo Ejecutar
1. Abre `index.html` en tu navegador.
2. Abre la consola del navegador (F12).
3. La aplicación se ejecutará automáticamente.
4. Sigue las instrucciones en los prompts.

## Validaciones
- Entradas numéricas se validan para evitar errores.
- División por cero está prevenida.
- Opciones de menú inválidas son manejadas.

## Tecnologías
- JavaScript (ES6+)
- HTML5
